# Parries

A parry is written as the matching target number plus `P`.

Examples:

| Symbol | Meaning |
|---|---|
| `1P` | Parry target 1 |
| `11P` | Parry target 11 |
| `3P` | Parry target 3 |

In a basic exchange, the defender responds to the same target line as the attack.

| Attack | Defense |
|---|---|
| `1` | `1P` |
| `11` | `11P` |
| `3` | `3P` |
