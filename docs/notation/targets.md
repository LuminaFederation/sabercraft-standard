# Targets

LUMINA uses a clock-face target system.

The target number represents the intended line or point of attack from the partner's perspective.

## Core targets

| Target | General idea |
|---:|---|
| 1 | High vertical / upper center line |
| 11 | Upper opposite line |
| 3 | Side line |
| 9 | Opposite side line |
| 5 | Lower line |
| 7 | Opposite lower line |

## Safety note

Targets indicate choreographed intent. They are not permission to strike the body.
